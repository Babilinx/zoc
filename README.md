# Zoc

Zoc is a block oriented, stack based concatenative programming language highly inspired by Zig.

# Functionalities
- Compiled
- Stack based
- Concatenative
- Strongly typed
- Errors as values
- Can call C function
- Programmed in itself
- No hidden allocations
- No hidden control flow
- Somewhat data oriented design
